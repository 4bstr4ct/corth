◆ _ParseOption

enum _ParseOption

Parsing flag to determine error level and behavior when parsing the arguments.

The strict parsing method will throw errors and complain if there are unknown flags/arguments. The lenient option will overlook all the errors listed above and focus only on trying to assign flags' values if they are present in command-line arguments list.
